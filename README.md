# omcp

API spec -> MCP server in 1 command

## Install

```
brew tap jruttan1/omcp
brew install omcp
```

## Usage

```
omcp
```

Point it at a YAML OpenAPI spec, select the endpoints you want to expose, and omcp starts an MCP server. Spin up a local API and use it in development, or point it at a production base URL and ship it.
